# Онар VK Mini App

VK Mini App - консультант по мерам социальной поддержки участников СВО и их семей в Республике Марий Эл.

## Технологии

- React 18 + TypeScript
- VKUI (официальные компоненты ВКонтакте)
- VK Bridge
- Vite

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

Приложение деплоится автоматически через Dokploy при пуше в main.
